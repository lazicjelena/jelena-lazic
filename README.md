## 🔬 Featured Project

### 🎙️ Serbian Emotional Speech Analysis

Analysis of emotional speech in Serbian using surprisal theory and machine learning models.

🔗 **Repository:** [link-to-your-repo]

---

### 📌 What this project does

* Extracts acoustic and linguistic features from speech
* Computes surprisal values using N-gram and transformer models
* Models relationship between surprisal and word duration
* Analyzes emotional speech patterns
